# Local final data

This directory is intentionally data-free in the public repository. The data-preparation notebook creates the merged training and held-out files consumed by `notebooks/2_modeling.ipynb`.

Final participant-level files must remain local and ignored. Do not commit, attach, archive, mirror, sample, or contribute them through issues or pull requests. They remain governed by the original provider terms and are not covered by the repository's software licence.

See [Dataset setup](../docs/DATASET_SETUP.md) for official access and local placement instructions.
