# Local processed data

This directory is intentionally data-free in the public repository. Running `notebooks/1_data_eploration.ipynb` with the required source files creates the processed training and held-out files used later in the workflow.

Processed participant-level files must remain local and ignored. Do not commit, attach, archive, mirror, sample, or contribute them through issues or pull requests. They remain governed by the original provider terms and are not covered by the repository's software licence.

See [Dataset setup](../docs/DATASET_SETUP.md) for official access and local placement instructions.
